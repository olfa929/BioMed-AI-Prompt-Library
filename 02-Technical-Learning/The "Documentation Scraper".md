
# The "Documentation Scraper"
**Category:** Technical Learning
**Task Description:** Turning a long manual or PDF into a set of actionable steps.

## 🤖 AI Prompt Template
> "I am attaching/pasting the documentation for [HARDWARE/SOFTWARE]. I have only 15 minutes to get this running. Extract: 1) The 'Quick Start' configuration, 2) Essential hardware pinouts/software dependencies, 3) The most critical safety warning, and 4) A 3-step 'Hello World' test to verify it is working."

## 🛠 Usage Instructions
- Paste the text of the manual or upload the PDF directly to the AI with this prompt.

**Estimated Time Savings:** 2 Hours (reading time).
