# The "Package Explorer" (R)
**Category:** Technical Learning
**Task Description:** Rapidly mastering a new R library (e.g., caret, ggplot2, tidyverse) for biomedical data.

## 🤖 AI Prompt Template
> "Act as a Senior Data Scientist proficient in R. I need to learn the [PACKAGE NAME] library for my biomedical engineering project. Provide: 1) The 5 most essential functions for data processing, 2) A code snippet demonstrating a standard workflow with this package, 3) Common errors to avoid when handling clinical datasets, and 4) A list of 3 high-quality documentation resources."

## 🛠 Usage Instructions
1. Replace `[PACKAGE NAME]` with the library you are struggling with.
2. If you have a specific dataset type (e.g., EEG signals), add "Context: I am working with [DATA TYPE]."

## 📝 Example Input & Output
- **Input:** "I need to learn the 'randomForest' package for heart disease classification."
- **Output:** A guide covering `randomForest()`, `importance()`, and cross-validation steps.

**Estimated Time Savings:** 60 Minutes
