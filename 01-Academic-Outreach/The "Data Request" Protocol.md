# Formal Medical Data Request
**Category:** Academic & Research Outreach
**Task Description:** Requesting access to specific datasets or clinical records for AI model training.

## 🤖 AI Prompt Template
> "Write a formal request to a [DATASET GATEKEEPER/INSTITUTION] asking for access to the [DATASET NAME]. Explain that I am a Biomedical Engineering student working on a project involving [SPECIFIC GOAL, e.g., identifying arrhythmias in ECG data]. Detail the technical stack (R/SVM) and assure them that the data will be used strictly for [ACADEMIC/NON-COMMERCIAL] purposes. Include a request for information on their data-sharing agreement (DSA) protocols."

## 🛠 Usage Instructions
- Use this when emailing hospital boards or open-science repository admins.
- Ensure you mention your University affiliation.
